Event Booking Management Chat Support

Description
The Event Booking Management Chat Support system is a web-based application that allows users to book events and interact with a chatbot for assistance. It provides an easy-to-use interface for managing bookings and improving user experience through chat support.

Technologies Used
- Python
- HTML
- (Flask - if used in backend)
- VS Code

How to Run the System
1. Clone the repository:
   git clone https://github.com/dxzhuofan/Event-Booking-Management-Chat-Support

2. Open the project in VS Code

3. Install requirements:
   pip install -r requirement.txt

4. Run the backend:
   python app.py

5. Open your browser and go to:
   http://localhost:5000
